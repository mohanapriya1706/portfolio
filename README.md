## ✨ Features

- **🌓 Dynamic Theming**: Seamlessly switch between Dark and Light modes with a custom-built Theme Provider.
- **🤖 AI Chatbot (Mona)**: Integrated AI assistant to answer professional queries about Mohana's experience and skills.
- **🎭 GSAP Animations**: High-performance, staggered entrance animations and smooth transitions.
- **🧠 Skills Visualization**: Categorized technical proficiency boards including MLOps, GenAI, and Google Cloud Platform.
- **📅 Interactive Timeline**: Collapsible professional experience entries for a clean and focused navigation experience.
- **📋 Contact Hub**: One-click email copying and direct `mailto:` integration to streamline communication.
- **📱 Responsive Design**: Fully optimized for desktop, tablet, and mobile viewing.

## 🛠️ Tech Stack

- **Frontend**: React 19 (Vite)
- **Styling**: Tailwind CSS 4.0
- **Animations**: GSAP 3 (ScrollTrigger)
- **Icons**: FontAwesome (`react-icons/fa`)
- **Backend (Chatbot)**: Cloud Run hosted AI API

## 🚀 Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohanapriya1706/portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run in development mode:
   ```bash
   npm run dev
   ```

## 📦 Deployment

The project is configured for **GitHub Pages** using GitHub Actions.

1. **Base URL**: Configured in `vite.config.js` to serve from `/portfolio/`.
2. **Automated Workflow**: Every push to the `main` branch triggers the `.github/workflows/deploy.yml` action, which builds and deploys the site automatically.
